# Desktop Wallpaper Changer

A Gui application to change the desktop wallpaper on Windows, Linux and Mac.

Build with
- [Electron](https://electronjs.org/)
- [React](https://reactjs.org/)
- [Unsplash API](https://unsplash.com/developers)

## Installation
```bash
npm install
npm start
```
